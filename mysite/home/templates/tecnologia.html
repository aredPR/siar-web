Componentes del Sistema

Visualiza todos los componentes hardware y software que conforman nuestro sistema.

Componentes IoT

Hardware

Arduino Uno

Microcontrolador que actúa como el cerebro del sistema IoT, procesando las señales de los sensores y controlando los actuadores según la lógica programada.

Llaveros RFID

Dispositivos pasivos con un chip y una antena que almacenan y transmiten datos de identificación de forma inalámbrica al acercarlos a un lector RFID.

Lector RFID

Dispositivo que emite señales de radio para detectar y leer la información almacenada en los tags RFID cuando estos se encuentran dentro de su rango de operación.

Protoboard

Herramienta para la conexión física entre componentes. Permite crear circuitos sin necesidad de soldadura, mientras que los cables establecen las conexiones eléctricas necesarias.

ESP32

Microcontrolador de bajo costo con Wi-Fi y Bluetooth integrado, ideal para proyectos de IoT que requieren conectividad inalámbrica y procesamiento potente.

Led rojo

Elemento de señalización que emite luz roja al recibir corriente eléctrica. Utilizado comúnmente para indicar estados o alertas en los sistemas IoT.

Resistencias 220Ω y 1kΩ

Componentes electrónicos que limitan el flujo de corriente en un circuito, protegiendo otros componentes y asegurando su correcto funcionamiento.

Power Bank

Batería portátil que proporciona energía para alimentar dispositivos IoT, especialmente en proyectos que requieren movilidad o no cuentan con una fuente de energía constante.

Cable entrada C

Cable utilizado para la conexión de dispositivos con puertos de entrada tipo C, comúnmente empleado en la carga o transferencia de datos.

Caja de contención

Contenedor que sirve para alojar y proteger los componentes electrónicos de posibles daños, además de organizar de manera ordenada el sistema IoT.

Tarjeta RFID

Dispositivo que almacena información de identificación en un chip, utilizado junto con los lectores RFID para el acceso o registro de datos.

Jumpers de conexión

Cables flexibles con conectores en ambos extremos, ideales para realizar conexiones rápidas y fáciles entre componentes en una protoboard o placa de pruebas.

Software

Arduino IDE

Entorno de desarrollo integrado utilizado para escribir, compilar y cargar el código en la placa Arduino. Facilita la programación y gestión del microcontrolador.

Bibliotecas RFID

Conjunto de funciones y clases predefinidas que simplifican la interacción con los lectores RFID, permitiendo leer y procesar la información de los tags de manera eficiente.

Librerías ESP32

Conjunto de funciones y clases predefinidas que simplifican el desarrollo de aplicaciones para el microcontrolador ESP32, facilitando su interacción con otros dispositivos y tecnologías.

Librerías Wi-Fi

Herramientas de software que facilitan la conexión y configuración de dispositivos IoT a redes Wi-Fi, permitiendo la comunicación de datos entre componentes.

Firebase Realtime Database

Base de datos en tiempo real que permite almacenar y sincronizar datos de forma instantánea entre dispositivos conectados, ideal para aplicaciones IoT que requieren actualizaciones en vivo.

Desarrollo Web

Frontend

Interfaz Web

Interfaz de usuario desarrollada con HTML, CSS y JavaScript, proporcionando un diseño atractivo y funcional que permite a los usuarios interactuar con el sistema a través de un navegador web.

Tailwind CSS

Framework de diseño de interfaz web que utiliza clases de utilidad para crear diseños personalizados de forma rápida y eficiente. Facilita el desarrollo de interfaces altamente responsivas y visualmente atractivas.

Backend

Google Authentication

Servicio de autenticación proporcionado por Google que permite a los usuarios iniciar sesión de manera segura y sencilla mediante sus cuentas de Google, mejorando la experiencia de usuario y la seguridad.

Python/Django

Framework de desarrollo web basado en Python, que ofrece una estructura robusta para crear aplicaciones web seguras, eficientes y escalables. Facilita la gestión del servidor, autenticación de usuarios y bases de datos.

Firebase (Web)

Plataforma de desarrollo que proporciona herramientas como base de datos en tiempo real, autenticación de usuarios y almacenamiento en la nube, facilitando la creación de aplicaciones web modernas y escalables.

Desarrollo Móvil

Frontend

Dart - Flutter

Lenguaje de programación desarrollado por Google, utilizado principalmente con Flutter para crear aplicaciones móviles y de escritorio de alto rendimiento con una única base de código.

Firebase Storage

Servicio de almacenamiento en la nube de Firebase que permite a las aplicaciones móviles guardar y gestionar archivos de forma segura, escalable y en tiempo real.

App Móvil

Backend

Flutter

Framework de UI de Google para crear aplicaciones nativas para móvil, web y escritorio desde una única base de código. Proporciona widgets personalizables para crear interfaces de usuario atractivas.

Firebase

Plataforma backend de Google que ofrece servicios como base de datos en tiempo real, autenticación de usuarios, notificaciones push y más, para aplicaciones móviles.

Google Authentication

Servicio de autenticación de Google que permite integrar el inicio de sesión en tus aplicaciones móviles utilizando las cuentas de Google de los usuarios de forma rápida y segura.

Dart

Uso combinado de Dart (para aplicaciones móviles) y Firebase (para backend) para crear aplicaciones móviles completas que gestionan datos, autenticación y almacenamiento en la nube.

Firebase Authentication

Servicio de autenticación de Firebase que permite gestionar el registro e inicio de sesión de los usuarios mediante múltiples métodos como email, Google, Facebook, etc.

Firebase Cloud Functions

Funciones en la nube de Firebase que permiten ejecutar código backend sin necesidad de administrar servidores, útil para realizar tareas como la gestión de notificaciones y la ejecución de procesos en segundo plano.
